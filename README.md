# WiFiScanner-MacOS

Command Line WiFi Scanner for MacOS (10.13)
![demo](https://raw.githubusercontent.com/darg20127/WiFiScanner-MacOS/master/Demo/demo.jpeg)

### Supports
- SSID
- PHY mode
- BSSID
- Channel
- Channel band
- Channel bandwidth
- Security
- Noise
- RSSI
